1 Dictionary

— Variable: *request*

Special variable to store Ningle Request, a instance of `<request>' in Ningle.Request package. Don't set to this variable directly. This is designed to be bound in lexical let.

— Variable: *response*

Special variable to store Ningle Response, a instance of `<response>' in Ningle.Response package. Don't set to this variable directly. This is designed to be bound in lexical let.

— Variable: *static-path*

url path for static files.

— Variable: *static-root*

Static file location.

— Variable: *webapp*

Ningle web app instance

— Macro: xdelete (url &key regexp) &body body

Create handler for delete requests.

— Macro: xget (url &key regexp) &body body

Create handler for get requests.

— Macro: xoptions (url &key regexp) &body body

Create handler for options requests.

— Macro: xpatch (url &key regexp) &body body

Create handler for patch requests.

— Macro: xpost (url &key regexp) &body body

Create handler for post requests.

— Macro: xput (url &key regexp) &body body

Create handler for put requests.

— Function: redirect url &key status

Redirect to url with status.

— Function: start &key server port productionp

Starts the server server on port port.

Set :productionp to t if you run this on your production server. Currently productionp has only effect on sbcl.

— Function: stop

Stops the server and halts the web application
